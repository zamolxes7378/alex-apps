Bernard
l'Hermite

We built a robot shaped like a hermit crab. It uses the Kitronik :MOVE Motor base and the BBC micro:bit board. We programmed it using MakeCode. We also laser-cut its shell and pincers to make it unique.

What Our Robot Does

radar

Ultrasonic Sensor

The ultrasonic sensor at the front lets the robot see walls to avoid crashes.

route

IR Line Following

The sensors under the robot read black lines to guide it automatically on the track.

code

MakeCode Programming

We write the program with MakeCode blocks. It's easy, visual, and very practical.
